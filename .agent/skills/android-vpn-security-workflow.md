# Skill: Android VPN Security Workflow

Trigger:
User asks about Android VPN, local proxy, runtime config, sanitizer, allow-lan, bind-address, ports, or auth.

Procedure:

1. Load Android security runbooks.
2. Locate sanitizer/runtime config path.
3. Ensure unsafe subscription values cannot override runtime safety.
4. Add or adjust tests if needed.
5. Validate generated runtime config.
6. Update `ERROR_HISTORY.md` if the pattern changed.
